# PCB Design Files

This directory contains the complete set of KiCad Printed Circuit Board (PCB) files, manufacturing outputs, and documentation required to view, modify, assemble, and produce this board.
---

## Directory Contents

Here is a quick overview of the essential files included in this folder:

| File / Folder Name | Type | Description |
| --- | --- | --- |
| **`interactive bom.html`** | HTML | Interactive Bill of Materials tool for easy manual assembly and component placement. |
| **`PCB BOM.csv`** | CSV | The complete, standard Bill of Materials listing all required electronic components, values, and footprints. |
| **`PCB_Schematic.pdf`** | PDF | A high-resolution PDF export of the circuit schematic for quick reference without needing CAD software. |
| **`KiCAD/production/WRO_Hat_V1.5.zip`** | ZIP Archive | The compiled Gerber and Drill files, ready to be uploaded to your favorite PCB manufacturer. |
| **KiCad Source Files** | KiCad | The raw `.kicad_pro`, `.kicad_sch`, and `.kicad_pcb` files for editing the design. |

## Detailed File Breakdown

### 1. The Interactive BOM (`interactive bom.html`)

This is a highly useful tool generated by the **InteractiveHtmlBom** plugin. If you plan on soldering this board yourself, simply open this file in any modern web browser. It provides a visual representation of the PCB, allowing you to highlight components, track your progress, and see exactly where each part belongs.

### 2. Standard Bill of Materials (`PCB BOM.csv`)

If you are ordering parts from suppliers like DigiKey, Mouser, or LCSC, this CSV file is your go-to resource. It contains a tabular breakdown of every resistor, capacitor, IC, and connector required.

### 3. Manufacturing Gerbers (`WRO_Hat_V1.5.zip`)

This ZIP archive contains all the industry-standard RS-274X Gerber files and Excellon drill files required by board houses (such as JLCPCB, PCBWay, or OSH Park).

* **Location:** Navigate to the `KiCAD/production/` folder to find this archive.
* **Usage:** Simply upload the `.zip` file directly to the manufacturer's quote page.

### 4. Schematic Reference (`PCB_Schematic.pdf`)

For software engineers writing firmware or hardware reviewers checking pinouts, the PDF schematic offers a clean, easily readable map of the entire circuit. It is fully searchable, making it easy to trace specific signals or power rails.

## Getting Started with KiCad

If you want to view or modify the raw board design:

1. Ensure you have the latest version of **KiCad EDA** installed on your system.
2. Clone or download this repository to your local machine.
3. Open the `.kicad_pro` project file in the main directory.
4. Launch the **Schematic Editor** or **PCB Editor** directly from the KiCad project manager.

---

> Important Note: Before sending the Gerber files for manufacturing, please verify that the design meets your specific board house's capabilities and tolerances (minimum trace width, hole size, and clearance).
