Electromechanical diagrams
====

`PCB V1.1 Schematic.pdf` is the PDF output of the schematic from KiCAD. It is the electrical schematic of the PCB used. Version 1.0 had an error (covered in `Engineering Manua/manual.pdf`. V1.1 is the fixed version.

The schematic is divided into 3 pages. The main page has the power connector and the power switch. The power page containst he circuitry for the buck converters. The logic page has the Raspberry Pi and the sensor connection.

The ROS2 node schematic will also be added here.
